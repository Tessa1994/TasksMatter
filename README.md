# TasksMatter
## Introduction
TasksMatter API aims to managment user and tasks(todos) data from mobile apps. 
The API is built in SpringMVC and Hibernate
## Quick start
Stack:
* JavaSe 1.8
* Spring 4.3.7
* Hibernate 5.0.12
* Mysql
.........
